# 快速上手

欢迎使用 **M記**。它把本地文件夹当作知识库，笔记就是普通的 `.md` 文件，不绑定云端账号，也不改成专有格式。

## 开始写作

1. 左侧文件树浏览知识库；用 **新建笔记**（默认 `Cmd/Ctrl+N`）开一篇新稿。
2. 工具栏可在 **编辑 / 分屏 / 预览** 之间切换（默认 `Cmd/Ctrl+3`）。分屏时左边改源码，右边看渲染。
3. 内容会自动保存；也可以随时手动保存（默认 `Cmd/Ctrl+S`）。

侧边栏搜索、当前文件查找、目录大纲等，可在 **设置 → 快捷键** 里查看或改绑。

## 小提示

- 粘贴图片会落到知识库里的附件目录，预览里直接可见。
- 需要导出 PDF、长图，或发布到博客 / 公众号草稿时，打开设置里对应入口即可。
- 示例笔记可以随便改；你改过的文件不会被后续同步覆盖。不需要的话，整份「示例笔记」文件夹删掉也行。
